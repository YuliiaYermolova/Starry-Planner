# 🌌 Starry Planner

Личный планер учёбы в стиле Ван Гога. Звёздное небо, цитаты философов, помодоро,
полная кастомизация под себя.

Все данные хранятся в браузере (`localStorage`) — приватно, без серверов и регистраций.

## Возможности

- **📊 Главная** — приветствие по времени суток, цитата дня (75+ кураторских + свои), статистика
- **📅 Расписание** — недельная сетка + прогресс учебного года
- **✅ Задачи** — фильтры, типы (домашка/проект/экзамен/чтение), дедлайны
- **🎯 Цели** — долгосрочные цели с подзадачами и автопрогрессом
- **📝 Заметки** — конспекты, ссылки, поиск
- **✨ Вдохновение** — цитаты, избранное, рекомендованные книги (Франкл, Юнг, Камю, Ницше, стоики), темы для самообучения
- **⏱ Фокус** — помодоро 25/5/15 с кольцом, циклами, статистикой по предметам
- **⚙️ Настройки** — профиль, 5 тем (Звёздная ночь, Подсолнухи, Оливки, Кафе, Ирисы), учебный год, предметы, экспорт/импорт JSON

## Локальный запуск

```bash
npm install
npm run dev
```

Открой http://localhost:3000

## Деплой на Vercel

1. Запушь репозиторий на GitHub
2. Зайди на https://vercel.com → **Add New → Project** → выбери репозиторий
3. Жми **Deploy** — ничего настраивать не нужно

## ⚠️ О хранении данных

Данные лежат **только в твоём браузере**. Если очистишь кэш, сменишь устройство
или браузер — потеряешь всё.

Делай регулярный бэкап через **Настройки → Экспорт JSON**. Сайт сам напомнит,
если не делал бэкап больше недели.
